# 📋 Documentação de Desenvolvimento - NeuroPlay

Esta pasta contém documentação técnica, relatórios de progresso e documentos de planejamento usados durante o desenvolvimento do projeto.

## 📁 Estrutura

```
development/
├── progress/           # Relatórios de progresso e sessões
├── planning/           # Planejamento e tarefas
├── architecture/       # Decisões arquiteturais (já existe em docs/)
├── testing/           # Estratégias e relatórios de teste
└── commits/           # Documentação de commits importantes
```

## 📝 Convenções

- Documentos de **progresso/sessão** vão em `progress/`
- Documentos de **planejamento** vão em `planning/`
