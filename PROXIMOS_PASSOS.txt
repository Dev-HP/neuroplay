PASSO 3: CONFIGURAR SENTRY (30 minutos)
===========================================

1. Criar conta: https://sentry.io

2. Criar projeto:
   - Nome: NeuroPlay
   - Plataforma: Flask + React
   - Copiar DSN

3. Adicionar secrets no GitHub:
   gh secret set SENTRY_DSN --body "https://xxx@sentry.io/xxx"
   gh secret set SENTRY_ORG --body "seu-org"
   gh secret set SENTRY_AUTH_TOKEN --body "seu-token"

4. Criar .env no backend:
   SENTRY_DSN=https://xxx@sentry.io/xxx
   SENTRY_ENVIRONMENT=production

5. Testar:

   - Gerar um erro proposital
   - Verificar no Sentry
